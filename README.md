# Elite Dangerous NEXT - Layout Manager v1.0.1-beta

**Automated keyboard layout switcher for Elite Dangerous**

---

## 🚀 Installation

1. **Download**: `EliteDangerous-LayoutManager-Setup-FIXED.exe`
2. **Run**: Double-click the installer
3. **Wait**: Installation takes 2-3 minutes (npm install + build)
4. **Launch**: Use desktop shortcut

---

## ✨ Features

- ✅ Automatic layout switching by active window
- ✅ Auto-start with Windows
- ✅ Hotkeys: Ctrl+Alt+R (Russian), Ctrl+Alt+E (English)
- ✅ Switch history viewer
- ✅ Statistics dashboard

---

## 📋 Requirements

- **OS**: Windows 10/11
- **Python**: 3.8+
- **Node.js**: 20.x
- **Disk Space**: ~200 MB

---

## 🎮 Quick Start

1. Launch **Layout Manager** from desktop
2. Add processes:
   - `EliteDangerous64.exe` → English
   - `COVAS NEXT.exe` → Russian
3. Click **"Start Monitor"**
4. Enable **"Auto-start"** for automatic launch
